# ridict

right click to dictionary

選択したテキストを右クリックし、ユーザーが設定した検索サイト（GET）を新規タブで開くブラウザ拡張機能。
Chrome と Firefox の両方に対応（同一フォルダ・同一コード）。
英和辞書向けに、選択範囲から半角英字のみを取り出して検索する。

## 構成

| ファイル | 役割 |
| --- | --- |
| `manifest.json` | 拡張の定義（権限・エントリポイント） |
| `background.js` | 右クリックメニューの登録とクリック処理（service worker） |
| `options.html` / `options.js` | 検索サイト URL・パラメータ名の設定画面 |

## インストール（開発用・未パッケージ）

### Chrome

1. `chrome://extensions` を開く
2. 右上の「**デベロッパー モード**」を ON
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. このフォルダ（`manifest.json` のある場所）を選択

### Firefox（121 以降）

1. `about:debugging#/runtime/this-firefox` を開く
2. 「**一時的なアドオンを読み込む**」をクリック
3. このフォルダの `manifest.json` を選択

※ Firefox の一時読み込みはブラウザを再起動すると消える。
恒久的に使うには addons.mozilla.org（AMO）での署名が必要（自己利用でも同様）。

## 使い方

1. 任意のページで英単語を範囲選択
2. 右クリック →「**「〇〇」を◯◯で調べる**」（辞書が 2 件以上あれば「**「〇〇」を辞書で調べる**」→ 辞書名を選択）
3. 選んだ検索サイトが新規タブで開く

## 設定

`chrome://extensions` の ridict →「詳細」→「拡張機能のオプション」、
または拡張アイコンの管理メニューから設定画面を開く。

辞書は最大 **5 件**まで登録でき、「メニューに表示」が ON のものが右クリックメニューに並ぶ。
1 件ごとに以下を設定する。

- **メニューに表示**: OFF にするとメニューから消えるが、設定は保持される
  （一時的に使わない辞書を取っておける。全部 OFF にするとメニュー自体が出ない）
- **表示名**: メニューに表示する辞書名（例: `英辞郎`）
- **検索サイトの URL**: クエリを除いたベース URL（例: `https://eow.alc.co.jp/search`）
- **クエリパラメータ名**: 検索語を渡すパラメータ名（例: `q`）

最終的に `ベースURL?パラメータ名=単語` の形で開く。
既定は英辞郎 on the WEB（`https://eow.alc.co.jp/search?q=単語`）。
